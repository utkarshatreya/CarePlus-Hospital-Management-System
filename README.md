# 🏥 CarePlus Hospital Management System

A modern Hospital Management System built using HTML, CSS, JavaScript, Node.js and JSON database.

## ✨ Features

- 👤 Patient Registration
- 📋 Patient Records Management
- 🔐 Admin Login
- 🕒 Live Date & Time
- 🚨 Emergency Contact Section
- 🎨 Modern Responsive UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- JSON Database

## 🚀 How to Run

1. Install Node.js
2. Open project folder
3. Install dependencies:
   npm install
4. Start server:
   node server.js
5. Open browser:
   http://localhost:3000
